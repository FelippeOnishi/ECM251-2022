Requisitos

- Conta:

```bash
Cadastrar email e senha ou logar com conta ja existente
Adicionar foto de perfil
Adicionar foto de fundo de perfil
Apelido (nickname)

...

```


- Jogos:

```bash
Adicionar os jogos (nome/foto/descrição)
Adicionar gameplay (vídeos sobre os jogos)
Sugestões de jogos de mesma caracteristica
Colocar o preco dos jogos
Promoçoes dos jogos
Data de lançamento
Categorias dos jogos
Classificação etaria
Requisitos do computador para rodar o jogo (minimo e máximo)
Informar qual a empresa desenvolvedora e qual a empresa distribuidora do jogo
Idiomas disponiveis
Extensões compraveis
Patch notes
Favoritar e adicionar na lista de desejos
Premios ganhos pelo jogo
Termos e condições
Plataforma utilizada
Modos de jogo (single player, multi player)
Disponibilidade em cada sistema operacional
Regiões disponiveis para ativação do jogo\
Quantidade de conquistas disponiveis
Status do jogo(Alpha, Beta)

...

```


-Plataformas:

```bash
Possibilidade de pesquisa de amigos e comunidades
Mudar idioma
Comunidade
Suporte
Link HOME
Opção de instalar o launcher da plataforma
Busca por jogos e suas categorias
Filtrar jogos pelo feedback
Ajuda

...

```

-Feedback:

```bash
Nome do usuario
Classificação de cada usuário(ex: 0-10)
Quantas horas foram jogadas
Texto explicando o motivo da avaliação
Possibilidade de classficar o feedback
Clasficicação média do jogo e quantas avaliações ele recebeu
Filtrar avaliações (Ex: Mostrar apenas positivas/negativas)

...

```


-Reembolso:

```bash
Motivo da solicitação
Qual jogo a ser reembolsado
Qual a solução o usuário deseja solicitar
Quais requisitos o usuario precisa ter para poder solicitar o reembolso (Quantidade maxima de horas, Quanto tempo o usuario comprou o jogo)
Verificação de que é o proprietário da conta que esta solicitando o reembolso
Tempo médio para a resposta a requisição

...

```


-Compra:

```bash
Saldo disponivel na conta
Métodos de pagamento
Carrinho de compra
Indicar qual o destino do jogo (Conta propria/presente)
Região da compra
Termos e condições para compra
Códigos de desconto

...

```
